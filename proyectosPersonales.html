Proyectos Personales

Se participó en el proyecto de creación de un software para desprendible de nómina para la DIVRI, donde estuve como Desarrollador de Front (React JS), otro de Base de datos (MySql) y yo de Backend (Spring Boot con Java 21). Donde se montó el servidor en Azure, con Linux Ubuntu, donde me tocó configurar el servidor e instalar git, Docker y Docker compose, montar Portainer para manejar los contenedores gráficamente, montar la base de datos de Mysql, GitLab community. Conectarme al servidor por SSH por mobaxterm y hacer el despliegue de la aplicación.
